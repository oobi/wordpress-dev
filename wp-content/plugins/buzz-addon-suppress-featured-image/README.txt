Adds a switch to suppress the featured image in the article view.



== Installation ==

This section describes how to install the plugin and get it working.

1. This is an add-on to "The Buzz". Make sure that the master plugin is installed and activated first.
1. Upload `buzz-addon-suppress-featured-image` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Tick the "suppress featured image" option in the Article Options metabox where required.

Requires specific theme support
