Buzz Blocks

== Description ==

Take newsletter layouts to the next level. This plugin adds a set of blocks to load Buzz Newsletter content into the block editor.

Support for full site editing as well as arbitrary configuration of each individual newsletter's layout.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/buzz-blocks` directory
1. Activate the plugin through the 'Plugins' screen in WordPress
