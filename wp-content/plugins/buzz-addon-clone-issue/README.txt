== Description ==

Adds the ability to clone an entire issue and associated articles to the Buzz Newsletter

== Installation ==

1. This is an add-on to "The Buzz". Make sure that the master plugin is installed and activated first.
1. Upload `buzz-addon-clone-issue` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
