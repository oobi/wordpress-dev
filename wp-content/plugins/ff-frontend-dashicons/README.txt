Load the Dashicons icon font on the front-end of your site, with shortcodes.

== Description ==
Makes the Dashicons CSS (font) available on the front end and in the WordPress editor.

Activate and you're ready to start using Dashicons on the front-end of your site via HTML and CSS.

Optionally use shortcode to insert an icon by itself:

[dashicon icon="format-quote"]

This will generate <span class="dashicons dashicons-format-quote"></span>

You can add additional classes or styles like so:

[dashicon icon="format-quote" class="my-custom-class" style="color:red; font-size:20px"]

If you don't want <span> then specify the tag like so:

[dashicon icon="format-quote" tag="i"]

This will generate <i class="dashicons dashicons-format-quote"></i>



== Installation ==
1. Install the Frontend Dashicons plugin either through your WordPress Plugins screen or by uploading the files to your server via FTP.
2. Activate Firefly Frontend Dashicons in the Plugins screen of your WordPress admin.
3. Begin using Dashicons icons through your site!
