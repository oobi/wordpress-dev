Add REST feed support to Envira Galleries (requires Envira plugin)

== Description ==

* Add REST feed support to Envira Galleries (requires Envira plugin).
* Feeds will be available like so:
** http://yoursite.com/wp-json/ff/v1/ff-envira 	: all Envira galleries
** http://yoursite.com/wp-json/ff/v1/ff-envira/123 : gallery with ID 123

== Installation ==

1. Make sure the Envira Gallery plugin is installed and activated
1. Upload the ff-envira-gallery-feed folder to your plugins directory
1. Activate the plugin through the 'Plugins' menu in WordPress
