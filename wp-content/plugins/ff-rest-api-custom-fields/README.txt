Custom post type, custom taxonomy and custom field support for WP REST API.

Also allows you to add custom fields to REST output.

== Description ==

Adds an interface to set avalability of your custom post types (or any custom post types registered) via the REST API.

Also allows you to add custom fields to REST output.

== Installation ==

1. Upload `ff-rest-api-custom-fields` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. access Firefly REST Custom settings to select what you want to make available to the API
