Add Links support to Midgard.

== Description ==

Define links to external pages and output as a JSON feed

== Installation ==

1. Make sure Midgard plugin is installed and active
1. Upload `midgard-links` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
