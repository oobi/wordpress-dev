Allows auto update for Firefly plugins and themes

== Description ==

Allows auto update for Firefly plugins for supported clients using the WordPress interface.

== Installation ==

1. Upload the plugin folder to wp-content/plugins
1. Activate or "Network activate" the plugin (on a multi site).
